# The Diataxis Compass

> Source: [diataxis.fr/compass](https://diataxis.fr/compass/)

The compass is a practical decision tool for classifying documentation content. It uses
two questions to identify which category content belongs to.

## The Two Dimensions

### Dimension 1: Action vs. Cognition

- **Action** — practical steps, doing things, hands-on
- **Cognition** — theoretical knowledge, understanding, thinking

### Dimension 2: Acquisition vs. Application

- **Acquisition** — study, learning, gaining new knowledge
- **Application** — work, using knowledge to accomplish goals

## The Decision Matrix

| | Acquisition (study) | Application (work) |
| --- | --- | --- |
| **Action** (practical) | Tutorial | How-to guide |
| **Cognition** (theoretical) | Explanation | Reference |

## How to Use the Compass

Ask two questions about any piece of documentation content:

1. **"Is this about action or cognition?"**
   - Does the reader do something (action) or learn/understand something (cognition)?

2. **"Is this about acquisition or application?"**
   - Is the reader studying/learning (acquisition) or working/applying (application)?

The intersection identifies the category.

## Multi-Scale Application

The compass works at multiple scales:

- **Individual sentences** — a single sentence might be explanation embedded in a how-to guide
- **Paragraphs** — a paragraph might shift from reference to tutorial mode
- **Sections** — a section might be pure reference within an otherwise mixed page
- **Entire documents** — the overall purpose of a document

## When to Use It

The compass is most valuable when you:

- Experience doubt about how to classify content
- Have difficulty writing — struggling to find the right voice or structure
- Notice content that feels "wrong" but can't articulate why
- Need to identify mixed content within a single page

It functions as a **course-correction tool**, not just a classification aid.
