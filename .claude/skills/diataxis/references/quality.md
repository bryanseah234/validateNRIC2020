# Documentation Quality

> Source: [diataxis.fr/quality](https://diataxis.fr/quality/)

Quality in documentation operates on two distinct levels: functional and deep.

## Functional Quality

Six dimensions that are mutually independent and objectively measurable:

| Dimension | Description |
| --- | --- |
| **Accuracy** | Is the information correct? |
| **Completeness** | Does it cover everything needed? |
| **Consistency** | Is it uniform in style, terminology, structure? |
| **Usefulness** | Does it serve the reader's actual needs? |
| **Precision** | Is it specific enough to be actionable? |
| **Redundancy tolerance** | Is unnecessary repetition avoided? |

These dimensions are independent — documentation can be accurate yet incomplete,
or complete yet useless.

## Deep Quality

Five markers that are subjectively judged and interdependent:

| Marker | Description |
| --- | --- |
| **Feeling good to use** | The reader's experience is pleasant and productive |
| **Having flow** | Content moves naturally without jarring transitions |
| **Fitting to human needs** | Structured around how people actually use docs |
| **Being beautiful** | Aesthetically coherent and well-crafted |
| **Anticipating the user** | Answers questions before they are asked |

Deep quality requires functional quality as a foundation but cannot be achieved
through measurement alone.

## How Diataxis Contributes to Quality

Diataxis cannot directly provide functional quality, but it facilitates both levels:

- **Exposes functional lapses** — structural clarity makes gaps and errors visible
- **Creates deep quality** — aligning content with user needs produces naturally
  flowing documentation
- **Preserves flow** — proper organization prevents jarring content-type shifts
- **Prevents rhythm disruption** — misplaced content (e.g., reference tables in a
  tutorial) breaks the reader's engagement

## Limitation

Diataxis can never be all that is required. It establishes the conditions for quality
documentation to emerge, but content accuracy, domain expertise, and good writing
remain the author's responsibility.
